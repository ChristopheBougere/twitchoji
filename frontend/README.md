# Twitchoji Frontend

The frontend is a React app.

## Install
```bash
npm install
cp .env_example .env
```
Fill the `.env` file with your parameters.

## Deploy
The frontend is deployed through Netlify when a PR merges on `master` branch.

## Lint
```bash
npm run lint
```

## Run locally
```bash
npm start
```

## Build production bundle
```bash
npm run build
```
